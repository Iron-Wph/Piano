# Example scores

## Beginner demo

The default demo is a simple two-hand teaching transcription of
“Twinkle, Twinkle, Little Star” in C major:

- `twinkle_twinkle_beginner.musicxml`
- `twinkle_twinkle_beginner.mid`

Both files contain the same 12-measure arrangement:

- right hand: the traditional melody;
- left hand: one whole-note bass pitch per measure;
- tempo: 96 quarter notes per minute;
- meter: 4/4.

### Source and rights

The melody is represented in the Library of Congress record:

> Dulcken, Ferdinand. *Twinkle, Twinkle, Little Star*. New York:
> Schuberth, Edw., 1879.

Source:
<https://www.loc.gov/item/2023832590/>

The Library of Congress identifies the containing collection as public domain
and free to use and reuse. The MusicXML and MIDI files in this directory are a
new, simplified transcription for Piano Hand; they are not copies of a
third-party MIDI or MusicXML arrangement.

The older `two_hand_scale.musicxml` file remains as a minimal parser fixture,
not as the default musical demonstration.
